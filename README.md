# Reduce-Mac-PDF-size-with-4D-Ghostscript-Plugin
This is a 4D v19.x sample to reduce 4D created PDF files with a 4D Ghostscript Plugin


<img width="848" alt="Bildschirmfoto 2023-03-29 um 18 32 06" src="https://user-images.githubusercontent.com/6436037/228607600-4cccc054-ec5d-45ac-a533-bd5703836e16.png">


Based on Keisuke Miyako GS Plugin, check there for updates. https://github.com/miyako/4d-plugin-gs

Under **Releases** the 4D (4th-Dimension) sample database is available.

The Ghostscript plugin from Miyako compresses a large Mac PDF file with various Ghostscript options.

## How to use the sample database ?

1. click on **Releases** and load the sample database
The sample database does not include the required Ghostscript plugin. 
To get the latest version of the GS plugin, I refer to the Github repository of Miyako.

<img width="600" alt="GS_download sample" src="https://user-images.githubusercontent.com/6436037/228828864-35574bfe-201a-41fa-877a-2b785e72ee26.png">


2. open this link to download the Ghostscript plugin from Miyako

https://github.com/miyako/4d-plugin-gs

3. click on **Releases**.

<img width="600" alt="GS_Miyako" src="https://user-images.githubusercontent.com/6436037/228828972-fbb394d9-2d81-41de-beae-992a35a63fe4.png">


4. download the Mac plugin image

<img width="600" alt="GS_Download" src="https://user-images.githubusercontent.com/6436037/228829025-82ae90bd-0624-4dad-b8e2-f370b705f205.png">


5. open the GS.dmg

6. drag the GS.bundle 4D plugin to the plugins folder of the sample application

<img width="985" alt="GS_drag" src="https://user-images.githubusercontent.com/6436037/228829073-faf9b0c0-bdb5-461a-bad4-bf5b2d274de4.png">


7. start the sample database with 4D >= v19.x and experience the magic of smaller PDF files on Mac

<img width="537" alt="start" src="https://user-images.githubusercontent.com/6436037/228829428-25bb698d-0a86-485e-b586-7b59968e8534.png">


**To create small PDF files with 4D Windows, I refer to this commercial plugin**

https://www.node.de/4d-plugin/4d-pdf-printer-plugin/
